# weicharts

使用简单的javascript代码使数据图形化。

## 创建图表

目前有饼图（PieChart）、序列图（SerialChart）。

````
var chart = new weicharts.PieChart("chartdiv");
````

## 数据

````
chart.setData(chartData);
````

## 尺寸

````
chart.setSize(500, 300);
````


## 样式

````
chart.setConfig(chartConfig);
````
