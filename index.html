VCharts

免费、易用的Web数据图表

Download this project as a .zip file Download this project as a tar.gz file

vcharts是一套JavaScript实现的Web图表库，由SVG+VML的技术实现。使用简单。

目前为测试阶段，饼图（PieChart）和序列图（SerialChart）已经可以使用，最新动态请关注微博@vcharts。

特点

  • 使用简单
  • 数据和显示逻辑分离
  • 自适应尺寸

使用方法

创建

vcharts按轴的不同分为以下几种图表

序列图 以一个序列为统一轴来展示数据，我们常见的柱状图、曲线图、面积图及其混合图都属于这种类型的图表。创建方法如下：

var mychart = new vcharts.SerialChart("mychartdiv");

饼图 描述单项占全部百分比，创建方法如下：

var mychart = new vcharts.PieChart("mychartdiv");
  • vcharts.SerialChart表示这是一个序列图表，
  • 参数是HTML中的div容器。这里是通过其id的方式，如："mychartdiv"；也可以通过元素对象的方式，不过要确保此元素载入之后才能访问。

方法

setData(data)

  • data：数据为json数组表示

return：该图表

setConfig(config)

  • config：数据为json数组表示

return：该图表

setSize(width, height)

  • width：设置图表宽度
  • height：设置图表高度

return：该图表

数据样例

使用json数组，饼图和单条序列图都可以使用如下格式：

[{label:"1月", value:"23"}, {label:"2月", value:"26"}, ...]

不同的图表可能在单元数据项中不太一样，如：

多条序列

｛label:"1月", value1:"22", value2:"33"｝

其它样例会随后面的版本一起推出，敬请关注

配置样例

这里我们完整的配置表，用于参考

饼图

{
    title : {
        name : "标题"
    },
    colors : ["#EE3300", "#FFD300", "#99DD33"],
    labelKey : "label",
    valueKey : "value"
}

序列图

{
    title : {
        name : "标题"
    },
    colors : ["#EE3300", "#FFD300", "#99DD33"],
    serialKey : "label",
    graphs : [{
        name : "销售量",
        valueKey : "value"
    }]
}

在下载到的压缩包中，我们将具体的实例放到了samples文件夹中。建议通过那里进行可视化图形来掌握配置信息。

关于尺寸

vcharts自动按优先级接受以下三种方式设置的尺寸

  • 优先级最高的是通过 setSize(width, height) 设置

  • 若未使用此方法设置，则读取css样式上的尺寸。

  • 若css样式也未设置，则使用默认大小（400*300）。
